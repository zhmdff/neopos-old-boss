# NeoPos - Boss Panel & Landing

A React-based management dashboard and marketing platform for the NeoPos ecosystem.

## Project Structure
- `src/Boss`: Management dashboard for business owners (reports, staff management, settings).
- `src/Landing`: Public-facing marketing and landing pages.
- `src/QR`: Customer-facing QR ordering system.
- `src/api`: Centralized axios configuration and API services.
- `src/utils`: Business logic utilities, helpers, and auth guards.

## Tech Stack
- **React 19** + **Vite 7**
- **Tailwind CSS 4**
- **SignalR**: Real-time business notifications.
- **Recharts**: Data visualization and business analytics.
- **Electron**: Wrapper for desktop deployment.

## Scripts
- `npm run dev`: Start development server.
- `npm run build`: Build for production.
- `npm run electron`: Launch the application in Electron.
